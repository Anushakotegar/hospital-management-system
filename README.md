# 🏥 Hospital Management System

## Project Overview

This project is a Hospital Management System developed using:

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MySQL (Hospitaldatab)

The system demonstrates database operations through a user-friendly web interface.

## Features

### 1. Insert Operation
- Add new patient records.
- Stores patient information in MySQL database.

### 2. Update Operation
- Update existing patient details using Patient ID.

### 3. Delete Operation
- Delete patient records from the database.

### 4. Display Records
- Retrieve and display all patient records in a table format.

### 5. Execute Stored Procedure
- Generate patient bills using a MySQL stored procedure.

### 6. Trigger Demonstration
- Database trigger executes automatically when specified database operations occur.

## Patient Details Managed

- Patient ID
- Name
- Age
- Gender
- Phone Number
- Address
